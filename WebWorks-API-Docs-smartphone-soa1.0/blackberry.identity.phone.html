blackberry.identity.phone.getLineIds

static Number[] getLineIds()

Supported Platform(s)
 - BlackBerry OS 5.0+

Description

Returns

blackberry.identity.phone.getLineLabel

static String getLineLabel(lineId : Number)

Supported Platform(s)
 - BlackBerry OS 5.0+

Description
 Returns a string description of the line specified.

Returns


Parameter Type Description
lineId Number [optional] ID of the line to retrieve the label for. If not specified the label for the default line is returned.

http://localhost:8472/blackberry/identity/phone/get

Description
 

Supported Platform(s)
 - BlackBerry OS 5.0+

Returns
{
    data : {
        "getLineIds" : [ 1 ],            // all the lines registered with the device.
        "getLineLabel" : "Line 1",       // description of default line
        "getLineNumber" : "15198887465", // number of default line
        "getLineType" : 1                // type of the default line specified: 1 - mobile, 2 - PBX
    }
}
