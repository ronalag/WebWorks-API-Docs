blackberry.utils.blobToString

static String blobToString(blob : Blob, [encoding: String])

Supported Platform(s)
 - BlackBerry OS 5.0+

Description
 Construct a new String by converting the blob using the specified character encoding

Returns
The String result from converting the Blob

Parameter Type Description
blob Blob The Blob object to be converted.
encoding String
Optional 		[Default Value: ISO-8859-1]

The name of a supported character encoding.BlackBerry supports the following character encodings: ISO-8859-1, UTF-8, UTF-16BE, US-ASCII. Besides these encodings, Web API supports also supports BASE64 encoding. Support of other encodings depends on the configuration of the BlackBerry PlayBook.
