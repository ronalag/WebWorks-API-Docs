HTML5 SQLError

Errors in the asynchronous database API are reported using callbacks that have a SQLError object as one of their arguments.

 - Ripple Emulator
SYNTAX_ERR = 5 Number The statement failed because of a syntax error, or the number of arguments did not match the number of ? placeholders in the statement, or the statement tried to use a statement that is not allowed, such as BEGIN, COMMIT, or ROLLBACK, or the statement tried to use a verb that could modify the database but the transaction was read-only.
 - BlackBerry OS 5.0+
 - BlackBerry PlayBook 1.0+
 - Ripple Emulator

 - Ripple Emulator
