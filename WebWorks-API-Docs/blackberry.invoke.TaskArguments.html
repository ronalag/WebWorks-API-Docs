TaskArguments

The TaskArguments object is an instance object, and is used as a parameter to the invoke() method when invoking the Tasks application.

Supported Platform(s)

blackberry.invoke.TaskArguments

blackberry.invoke.TaskArguments ([task: blackberry.pim.Task])

Supported Platform(s)

 - BlackBerry OS 5.0+
 - Ripple Emulator

Description

The TaskArguments object must be created as an instance using the new keyword.



Parameter Type Description
task blackberry.pim.Task
Optional 		Task to view in Tasks application.

Code Example(s)

<script type="text/javascript">
  var task= new blackberry.pim.Task();
  task.summary = 'Pick it up';
    
  var args = new blackberry.invoke.TaskArguments(task);
  args.view = 0;
   
  blackberry.invoke.invoke(blackberry.invoke.APP_TASKS, args);    
</script>
