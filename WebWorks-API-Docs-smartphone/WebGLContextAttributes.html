WebGLContextAttributes

The WebGLContextAttributes interface contains drawing surface attributes and is passed as the second parameter to getContext

Properties:

Property Type Description Supported Platform(s)
antialias
default: true 		boolean If the value is true and the implementation supports antialiasing the drawing buffer will perform antialiasing using its choice of technique (multisample/supersample) and quality. If the value is false or the implementation does not support antialiasing, no antialiasing is performed
 - BlackBerry PlayBook 2.0
 - Ripple Emulator
depth
default: true 		boolean If the value is true, the drawing buffer has a depth buffer of at least 16 bits. If the value is false, no depth buffer is available
 - BlackBerry PlayBook 2.0
 - Ripple Emulator
premultipliedAlpha
default: true 		boolean If the value is true the page compositor will assume the drawing buffer contains colors with premultiplied alpha. If the value is false the page compositor will assume that colors in the drawing buffer are not premultiplied
 - BlackBerry PlayBook 2.0
 - Ripple Emulator
stencil
default: false 		boolean If the value is true, the drawing buffer has a stencil buffer of at least 8 bits. If the value is false, no stencil buffer is available
 - BlackBerry PlayBook 2.0
 - Ripple Emulator
