HTMLTrackElement

The track element allows authors to specify explicit external timed text tracks for media elements. It does not represent anything on its own.

There must not be two track element children of the same media element whose kind attributes are in the same state, whose srclang attributes are both missing or have values that represent the same language, and whose label attributes are again both missing or both have the same value.

Property Type Description Supported Platform(s)
default Boolean The default attribute, if specified, indicates that the track is to be enabled if the user's preferences do not indicate that another track would be more appropriate. There must not be more than one track element with the same parent node with the default attribute specified.
 - BlackBerry OS 6.0+
 - BlackBerry PlayBook

 - BlackBerry OS 6.0+
 - BlackBerry PlayBook

The value of the label attribute, if the attribute is present, must not be the empty string.
 - BlackBerry OS 6.0+

 - BlackBerry OS 6.0+
