document.createElement("audio")

Supported Platform(s)

 - BlackBerry OS 6.0+
 - BlackBerry PlayBook 1.0+

Description

Creates an HTMLAudioElement



Code Example(s)

var audio = document.createElement('audio');
audio.src = 'audio.wav';
audio.controls = true;
document.body.appendChild(audio);
